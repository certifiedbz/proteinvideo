Transcription, Translation, and About the website

Located in the Nucleus you will find mRNA starting the process of transcription. With the help of the RNA polymerase, the DNA strand is unzipped. This allows for the mRNA to make a complementary sequence to the DNA strand with the change of thymine to uracil. After the single-stranded mRNA is complete it travels from the Nucleus to the Cytoplasm to continue with Translation. During the process of translation, the mRNA is put into codons which is a sequence of 3 nitrogen bases which is the code that tRNA uses to construct amino acids. The tRNA with the use of codons and anticodons constructs polypeptides which are strands of amino acids. We can predict what amino acids are being formed with the use of only DNA strands. This website that I have created allows us to be able to predict which amino acids are being created with the use of a DNA strand codon.
